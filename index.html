Airtable Setup Guide

Follow these 5 steps to enable real-time sign-ups visible to everyone on the team:

  1. Go to airtable.com → create a free account
  2. Create a new Base called "PFR Signups"
  3. Create a table called "Signups" with 4 fields (all Single Line Text):
    SessionId  ·  PlayerName  ·  SessionDate  ·  SignedUpAt
  4. Go to airtable.com/create/tokens → create a Personal Access Token
    Scopes needed: data.records:read and data.records:write
  5. Open pfr-team-site.html in any text editor and replace the two values at the top of the <script> block: AIRTABLE_TOKEN = 'paste your token here'
    AIRTABLE_BASE_ID = 'paste your base ID here'     Your Base ID is in the URL when you open your base:
    airtable.com/appXXXXXXXXXX/tbl.../viw...

Once configured, every sign-up is saved instantly and visible to all teammates in real time. The table in Airtable also gives you a clean list of who's coming to each session.
